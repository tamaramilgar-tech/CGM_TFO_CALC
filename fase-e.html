FASE E

Importación y exportación de hojas de cálculo

Intercambio de archivos entre Calc y otras aplicaciones o formatos.

Resumen de la fase

Una empresa recibe listados de productos en CSV y después debe compartir el resultado con otra oficina que usa Excel.

Material necesario

LibreOffice CalcUn archivo CSV de ejemploCapacidad para guardar como ODS y XLSX

Conceptos básicos que debes conocer

  • ODS es el formato nativo de LibreOffice Calc.
  • XLSX es un formato habitual de Microsoft Excel.
  • CSV guarda datos separados por delimitadores y no conserva formato avanzado.
  • Importar permite abrir o incorporar datos de otros formatos.
  • Exportar sirve para compartir el archivo en un formato compatible con otros programas.
Idea clave: antes de empezar la práctica, conviene que el estudiante comprenda para qué sirve cada herramienta y en qué situación real podría utilizarla.

Práctica guiada paso a paso

Paso 1. Crear un CSV de ejemplo o abrir uno existente

Guarda el archivo como fase-e-importacion-exportacion.ods y trabaja también con un pequeño CSV con columnas de código, artículo y stock.

Paso 2. Importar el archivo CSV

Desde Calc, abre el CSV y observa la ventana de importación. Revisa el separador y la vista previa antes de aceptar.

Paso 3. Comprobar los datos importados

Verifica que cada campo aparece en su columna correcta y que números y textos se han interpretado bien.

Paso 4. Guardar en formato ODS

Una vez abierto o adaptado el contenido, guarda una copia en formato .ods para seguir trabajando con todas las opciones de Calc.

Paso 5. Exportar o guardar como XLSX

Utiliza Guardar como y selecciona el formato de Excel para generar una versión compatible con otras aplicaciones.

Paso 6. Comparar formatos

Pide al alumnado que observe qué elementos se mantienen y cuáles pueden cambiar según el formato elegido.

Paso 7. Añadir una pequeña modificación

Cambia un valor de stock, añade una fila nueva y vuelve a exportar el archivo para consolidar el proceso.

Paso 8. Reflexionar sobre compatibilidad

Explica cuándo conviene usar ODS, cuándo XLSX y cuándo CSV según el objetivo del documento.

Tipo test final

Este pequeño cuestionario sirve para comprobar si se han entendido los conceptos trabajados en la práctica.

1. ¿Cuál es el formato nativo de Calc?

Opciones: ODS / CSV / DOCX

Respuesta correcta: ODS.

2. ¿Qué formato se usa mucho para intercambiar datos simples separados por comas o punto y coma?

Opciones: CSV / PNG / PDF

Respuesta correcta: CSV.

3. ¿Qué acción permite compartir el archivo con usuarios de Excel?

Opciones: Guardar como XLSX / Cambiar la fuente / Insertar una macro

Respuesta correcta: Guardar como XLSX.

4. ¿El CSV conserva siempre gráficos y formato complejo?

Opciones: Sí / No / Solo en línea

Respuesta correcta: No.

5. ¿Qué conviene revisar al abrir un CSV?

Opciones: El separador y la vista previa / El color del escritorio / La impresora

Respuesta correcta: El separador y la vista previa.
