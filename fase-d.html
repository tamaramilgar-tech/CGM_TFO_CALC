FASE D

Macros para automatizar tareas

Uso de macros para la realización de documentos y plantillas.

Resumen de la fase

En una oficina se necesita aplicar siempre el mismo formato a tablas de pedidos. Para ahorrar tiempo, se decide grabar una macro.

Material necesario

LibreOffice CalcArchivo con una tabla simpleOpciones de seguridad de macros revisadas si fuese necesario

Conceptos básicos que debes conocer

  • Una macro es una secuencia de acciones grabadas para automatizar tareas repetitivas.
  • Calc permite grabar macros sencillas si la opción está habilitada.
  • Las macros pueden aplicarse a formato, inserción de textos y acciones repetidas.
  • Es importante guardar y probar una macro antes de usarla muchas veces.
  • También pueden asociarse a botones o accesos rápidos.
Idea clave: antes de empezar la práctica, conviene que el estudiante comprenda para qué sirve cada herramienta y en qué situación real podría utilizarla.

Práctica guiada paso a paso

Paso 1. Crear la tabla base

Guarda el archivo como fase-d-macros.ods. Introduce una tabla sencilla con encabezados y datos.

Paso 2. Planificar la tarea repetitiva

Decide qué acciones vas a grabar: poner los encabezados en negrita, aplicar color de fondo, centrar texto y añadir borde a la tabla.

Paso 3. Grabar la macro

Ve a Herramientas > Macros > Grabar macro. Realiza las acciones previstas sobre la tabla y detén la grabación.

Paso 4. Guardar la macro

Pon un nombre claro, por ejemplo FormatoTablaPedidos, y guárdala en el documento actual.

Paso 5. Probar la macro

Crea una segunda tabla pequeña y ejecuta la macro para comprobar que repite el formato automáticamente.

Paso 6. Insertar un botón o acceso

Si se desea, inserta un botón de formulario y asígnale la macro para que el alumnado vea una forma práctica de ejecutarla.

Paso 7. Comentar ventajas y límites

Explica que las macros son útiles para tareas repetitivas, pero conviene revisarlas y usarlas con cuidado, especialmente si vienen de otros archivos.

Paso 8. Guardar y cerrar

Guarda el documento verificando que la macro queda disponible al volver a abrir el archivo.

Tipo test final

Este pequeño cuestionario sirve para comprobar si se han entendido los conceptos trabajados en la práctica.

1. ¿Qué es una macro?

Opciones: Un gráfico / Una secuencia automatizada de acciones / Un tipo de fórmula

Respuesta correcta: Una secuencia automatizada de acciones.

2. ¿Para qué sirve en esta práctica?

Opciones: Automatizar el formato de tablas / Borrar hojas / Conectarse a internet

Respuesta correcta: Automatizar el formato de tablas.

3. ¿Es recomendable poner nombres claros a las macros?

Opciones: No / Sí / Solo en Excel

Respuesta correcta: Sí.

4. ¿Se puede asignar una macro a un botón?

Opciones: Sí / No / Solo en Writer

Respuesta correcta: Sí.

5. ¿Conviene revisar macros de archivos externos?

Opciones: No hace falta / Sí, por seguridad / Solo si son largas

Respuesta correcta: Sí, por seguridad.
