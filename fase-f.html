FASE F

Calc como base de datos

Formularios, listas, filtrado, protección y ordenación de datos.

Resumen de la fase

Debes gestionar una lista de clientes de una academia y localizar rápidamente información por localidad, fecha de alta o estado de pago.

Material necesario

LibreOffice CalcTabla de clientesConocimientos básicos sobre ordenar y seleccionar rangos

Conceptos básicos que debes conocer

  • Una hoja de cálculo puede almacenar listas estructuradas como una base de datos simple.
  • Cada fila suele representar un registro y cada columna un campo.
  • Ordenar organiza los datos según un criterio.
  • Filtrar muestra solo la información que cumple determinadas condiciones.
  • La protección ayuda a evitar cambios accidentales en celdas o hojas.
Idea clave: antes de empezar la práctica, conviene que el estudiante comprenda para qué sirve cada herramienta y en qué situación real podría utilizarla.

Práctica guiada paso a paso

Paso 1. Crear la lista de datos

Guarda el archivo como fase-f-base-datos.ods. Introduce una tabla con encabezados claros y varios registros de clientes.

Paso 2. Convertir la tabla en una lista usable

Asegúrate de que no haya filas ni columnas vacías intermedias. Esto facilita ordenar y filtrar correctamente.

Paso 3. Ordenar los registros

Selecciona la tabla y usa la opción de ordenar por Localidad o por Fecha alta para ver cómo cambia el orden de los datos.

Paso 4. Aplicar filtros

Activa el filtro automático y muestra únicamente los clientes de una localidad concreta o con estado Pendiente.

Paso 5. Crear una vista útil

Combina filtro y ordenación para localizar, por ejemplo, los clientes pendientes de Santa Cruz.

Paso 6. Proteger celdas o la hoja

Protege la hoja o ciertas celdas para evitar modificaciones accidentales en encabezados o fórmulas.

Paso 7. Comentar el uso de formularios

Explica que Calc puede servir como base para introducir datos ordenados y que existen formularios sencillos o herramientas relacionadas para facilitar la entrada.

Paso 8. Guardar la práctica

Revisa el funcionamiento de la lista y comprueba que sabes quitar y volver a aplicar los filtros.

Tipo test final

Este pequeño cuestionario sirve para comprobar si se han entendido los conceptos trabajados en la práctica.

1. En una lista de datos, ¿qué suele representar cada fila?

Opciones: Un registro / Un gráfico / Una contraseña

Respuesta correcta: Un registro.

2. ¿Para qué sirve un filtro?

Opciones: Para mostrar solo los datos que cumplen una condición / Para traducir textos / Para insertar imágenes

Respuesta correcta: Para mostrar solo los datos que cumplen una condición.

3. ¿Qué permite la ordenación?

Opciones: Cambiar el orden de los registros según un criterio / Bloquear el archivo / Crear macros

Respuesta correcta: Cambiar el orden de los registros según un criterio.

4. ¿Por qué puede ser útil proteger una hoja?

Opciones: Para evitar cambios accidentales / Para sumar más rápido / Para importar un CSV

Respuesta correcta: Para evitar cambios accidentales.

5. ¿Puede una hoja de cálculo funcionar como una base de datos simple?

Opciones: Sí / No / Solo con internet

Respuesta correcta: Sí.
