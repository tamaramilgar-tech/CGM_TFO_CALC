## Publicación rápida en GitHub Pages
1. Crea un repositorio nuevo en GitHub.
2. Sube todos estos archivos a la raíz del repositorio.
3. Entra en `Settings > Pages`.
4. En `Build and deployment`, selecciona `Deploy from a branch`.
5. Elige la rama `main` y la carpeta `/root`.
6. Guarda los cambios.
7. Espera unos segundos y abre la URL pública que genere GitHub Pages.

## Personalización
- Cambia colores y estilos en `style.css`.
- Edita textos y prácticas en cada archivo `fase-*.html`.
- Puedes renombrar el proyecto y ajustar la portada en `index.html`.
