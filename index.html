Presentación

Objetivo del recurso

Qué va a aprender el alumnado

El alumnado trabajará con una hoja de cálculo de forma progresiva: introducirá distintos tipos de datos, aplicará referencias y rangos, construirá fórmulas y funciones, generará gráficos, automatizará tareas con macros, intercambiará archivos con otros formatos y usará Calc para gestionar datos como si fuese una pequeña base de datos.

Cómo está organizada la web

Cada fase incluye tres bloques: conceptos básicos, práctica guiada y tipo test. La idea es que el estudiante comprenda primero lo esencial y después desarrolle una tarea realista siguiendo instrucciones claras y ordenadas.

Itinerario didáctico

Las 6 fases de aprendizaje

Cada fase trabaja uno de los aspectos clave del manejo de Calc en el módulo.

Fase A

Tipos de datos y referencias

Uso de datos, celdas, rangos, hojas, libros y referencias relativas, absolutas y mixtas.

 Fase B

Fórmulas y funciones

Operaciones, funciones básicas y condicionales aplicadas a un caso práctico de notas.

 Fase C

Gráficos

Creación e interpretación de gráficos de barras, líneas y sectores a partir de datos reales.

 Fase D

Macros y automatización

Grabación de macros sencillas para automatizar tareas repetitivas y mejorar la productividad.

 Fase E

Importación y exportación

Trabajo con CSV, XLSX y ODS para importar, adaptar y compartir hojas de cálculo.

 Fase F

Calc como base de datos

Listas, filtrado, ordenación, formularios sencillos y protección de datos en una tabla.

Aplicación

Uso didáctico recomendado

En clase

Presentar la fase, explicar los conceptos y resolver la práctica con acompañamiento inicial.

Trabajo autónomo

El alumnado puede seguir los pasos de forma individual y comprobar lo aprendido con el test final.

Evaluación

Se puede valorar el resultado final de la práctica, la autonomía en el proceso y la corrección del test.
