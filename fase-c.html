FASE C

Gráficos en Calc

Creación y modificación de gráficos de distintos tipos.

Resumen de la fase

Una tienda quiere mostrar la evolución de sus ventas mensuales y la distribución de productos vendidos para presentar los datos en una reunión.

Material necesario

LibreOffice CalcTabla con meses y ventasTabla con categorías y porcentajes

Conceptos básicos que debes conocer

  • Un gráfico representa visualmente datos numéricos.
  • Los tipos más frecuentes en esta práctica son barras, líneas y sectores.
  • Un buen gráfico necesita título, leyenda y datos claros.
  • Cada tipo de gráfico sirve mejor para una finalidad distinta.
  • Modificar un gráfico ayuda a hacerlo más comprensible.
Idea clave: antes de empezar la práctica, conviene que el estudiante comprenda para qué sirve cada herramienta y en qué situación real podría utilizarla.

Práctica guiada paso a paso

Paso 1. Crear la tabla de ventas

Guarda el archivo como fase-c-graficos.ods. Introduce los meses y las ventas mensuales en dos columnas.

Paso 2. Insertar un gráfico de barras

Selecciona la tabla completa. Ve a Insertar > Gráfico y elige un gráfico de barras. Finaliza el asistente.

Paso 3. Editar el título del gráfico

Haz doble clic sobre el gráfico y cambia el título por Ventas mensuales del primer trimestre ampliado o uno similar.

Paso 4. Modificar elementos visuales

Ajusta la leyenda, el color de las barras y el tamaño del gráfico para mejorar la presentación.

Paso 5. Crear un gráfico de líneas

Duplica o reutiliza la misma tabla para insertar ahora un gráfico de líneas y comparar cómo se ve la evolución de los datos.

Paso 6. Añadir un gráfico circular

Crea una segunda tabla con categorías de productos y cantidades. Inserta un gráfico de sectores para representar porcentajes o distribución.

Paso 7. Comparar tipos de gráfico

Pide al alumnado que explique qué gráfico resulta más útil para evolución temporal y cuál para reparto porcentual.

Paso 8. Guardar la práctica

Revisa que los gráficos sean legibles, estén bien titulados y guarden relación con los datos.

Tipo test final

Este pequeño cuestionario sirve para comprobar si se han entendido los conceptos trabajados en la práctica.

1. ¿Qué gráfico suele ser adecuado para comparar cantidades entre categorías?

Opciones: Barras / Sectores 3D siempre / Ninguno

Respuesta correcta: Barras.

2. ¿Qué gráfico es útil para ver una evolución en el tiempo?

Opciones: Líneas / Sectores / Tabla dinámica

Respuesta correcta: Líneas.

3. ¿Qué gráfico suele utilizarse para mostrar porcentajes de un total?

Opciones: Circular o de sectores / Dispersión / Columnas vacías

Respuesta correcta: Circular o de sectores.

4. ¿Qué elemento ayuda a identificar el contenido del gráfico?

Opciones: El título / La contraseña / La macro

Respuesta correcta: El título.

5. ¿Se puede modificar un gráfico una vez creado?

Opciones: No / Sí / Solo exportándolo

Respuesta correcta: Sí.
